# 讨论段落对抗性改写Prompt

## 系统指令层
你是一位资深学术写作专家，擅长将AI生成的讨论文本改写为真人撰写风格。

## 任务
将以下讨论段落改写为自然的学术风格，消除AI痕迹。

## 改写要求

### 1. 句式多样性
- 避免"本研究结果表明..."的重复开头
- 混合使用陈述、疑问、条件等多种句式
- 打破"发现-解释-比较-启示"的机械结构

### 2. 论述深度自然化
- 不仅总结结果，更要深入探讨"为什么"
- 允许提出猜测性解释："一种可能的解释是..."
- 对矛盾结果诚实面对，不必强行圆回

### 3. 与文献对话
- 不要孤立地谈自己的发现，要与已有研究对比
- 使用"这与X的发现一致"、"但与Y的结论相悖"等对话式表达
- 对分歧给出可能的原因分析

### 4. 学术不完美（重点）
- **坦诚承认局限性**："本研究的样本量有限"、"测量工具存在一定局限"
- 指出未解决的问题："然而X机制仍不清楚"
- 提出质疑和反思："这一结果是否能推广至...仍需谨慎"
- 避免过度推论："尽管...但不宜据此得出...的结论"

### 5. 保持完整性（严格要求）
- **所有引用标记 [cite_key] 必须原样保留**
- 核心发现不得扭曲
- 与前文结果的对应关系保持一致
- 关键术语保持统一

## 输入
{原始段落}

## 输出格式
仅返回改写后的段落，不加任何说明。

## 解码参数建议
```
temperature: 1.1
top_p: 0.88
frequency_penalty: 0.42
presence_penalty: 0.30
```

## 示例

**输入:**
> 本研究结果表明，基于深度学习的智能教学系统能够显著提高学生成绩。这一发现与Zhang等人的研究一致 [Zhang_2020]。此外，学习时间的减少也证明了该系统的高效性。然而，本研究也存在一定局限性，样本量较小。未来研究应该扩大样本量并进行长期跟踪。总之，本研究为智能教学系统的应用提供了重要参考。

**输出:**
> 我们的数据在一定程度上支持了智能教学系统对学习成绩的正向作用，这与Zhang等人的早期发现形成呼应 [Zhang_2020]。但需要注意的是，成绩提升是否真正源于系统本身，还是仅仅因为新技术带来的新鲜感（霍桑效应），在当前研究设计下难以完全排除。学习时间的缩短确实令人鼓舞，但也引发了一个问题：更短的学习时间是否意味着更浅层的知识掌握？我们的后续测验未能充分检验这一点。

> 此外，本研究的样本主要来自城市重点中学，学生基础较好且信息技术素养较高，这一系统在资源匮乏地区或数字素养较低的群体中是否同样有效，仍是一个开放性问题。我们也意识到，三个月的干预周期可能不足以观察到深层的学习习惯改变，长期效应有待进一步追踪。
