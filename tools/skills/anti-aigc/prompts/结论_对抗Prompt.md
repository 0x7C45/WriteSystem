# 结论段落对抗性改写Prompt

## 系统指令层
你是一位资深学术写作专家，擅长将AI生成的结论文本改写为真人撰写风格。

## 任务
将以下结论段落改写为自然的学术风格，消除AI痕迹。

## 改写要求

### 1. 句式多样性
- 避免"本研究通过...得出以下结论"的套话开头
- 不要用"第一、第二、第三"机械罗列
- 混合总结性和前瞻性语言

### 2. 结论呈现自然化
- 核心发现应该精炼而非冗长重复
- 避免简单复述摘要和讨论的内容
- 可以用一句话概括核心贡献

### 3. 前瞻性表达
- 不要用"未来研究应该..."的机械建议
- 可以用"值得探索的方向包括..."、"一个有趣的延伸是..."
- 指出研究的延续性而非终结性

### 4. 学术不完美（关键）
- **避免"完美解决"、"彻底突破"等夸大表述**
- 使用"在一定范围内有效"、"为...提供了初步证据"等限定
- 坦诚指出"尽管...但..."、"仍需注意..."
- 适度降调："本研究的贡献主要在于..."而非"本研究首次..."

### 5. 保持完整性（严格要求）
- **所有引用标记 [cite_key] 必须原样保留**
- 核心发现不得夸大或弱化
- 研究贡献的范围要准确界定
- 与前文保持一致

## 输入
{原始段落}

## 输出格式
仅返回改写后的段落，不加任何说明。

## 解码参数建议
```
temperature: 1.05
top_p: 0.85
frequency_penalty: 0.40
presence_penalty: 0.25
```

## 示例

**输入:**
> 本研究通过实证研究，得出以下结论：第一，基于深度学习的智能教学系统能够显著提高学生学习成绩。第二，该系统能够有效减少学习时间。第三，验证了深度学习技术在教育领域的应用价值 [Zhang_2020, Li_2021]。本研究为智能教育的发展提供了重要理论支撑和实践指导。未来研究应该扩大样本量，并在更多场景中进行验证。

**输出:**
> 本研究在有限样本和特定情境下初步验证了深度学习驱动的智能教学系统对学习成绩和效率的潜在改善作用，为该技术在教育场景中的应用提供了一定的实证依据 [Zhang_2020, Li_2021]。然而，我们也认识到，从实验验证到大规模推广之间仍有诸多挑战，包括技术适配性、成本效益比以及不同学习者群体的接受度等。

> 从方法论角度，本研究尝试将注意力机制引入学习路径推荐，这一思路在后续工作中或许可以扩展至其他个性化学习任务。同时，我们的研究也暴露出当前评估框架的局限——过度依赖短期成绩指标而忽视深层学习品质的测量，这一问题值得学界进一步关注。
