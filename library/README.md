## 主要内容

### 1. 组织方式（按主题分类）

定义了清晰的目录结构，按以下主题分类：
- **writing-theory/** - 写作理论（创意、技术、学术、商务写作）
- **linguistics/** - 语言学（句法、语义、语用、话语分析）
- **cognitive-science/** - 认知科学（记忆、注意力、学习、创造力）
- **nlp-ai/** - 自然语言处理与AI（语言模型、文本生成、评估、应用）
- **domain-knowledge/** - 领域知识（科学、技术、人文、社会科学）
- **reference/** - 参考资料（风格指南、术语表、模板库）

每个主题下包含 `index.md` 元数据索引和 `notes/` 笔记子目录。

### 2. 文件命名规范

**基本格式**：`[年份]-[作者]-[简短标题].[格式]`

**规则细节**：
- 年份：4位数字（如 2024）
- 作者：姓氏拼音或英文（如 smith、zhang）
- 标题：3-5个关键词，连字符连接
- 特殊前缀：`review-`（综述）、`classic-`（经典文献）

**示例**：
- `2024-smith-cognitive-load-writing.pdf`
- `2024-review-johnson-nlp-advances.pdf`
- `1980-classic-flower-cognitive-process-writing.pdf`

### 3. 使用指南

**完整操作流程**：

1. **添加新文献**：重命名 → 放入对应目录 → 更新索引 → 添加笔记
2. **检索文献**：按主题浏览、全文检索、按作者/年份检索
3. **引用文献**：标准化引用格式，使用 `[library:文件名]` 标记
4. **创建阅读笔记**：提供完整笔记模板（核心观点、研究方法、批判性思考、项目启示）
5. **定期维护**：每周/每月/每季度维护检查清单
